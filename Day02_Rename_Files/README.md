# Day 02 - Python File Renamer

## Problem
Renaming many files manually takes time.

## Solution
Python script automatically renames files with sequential numbering.

## Features
- Rename multiple files instantly
- Custom file name prefix
- Sequential numbering

## Example

Before:
IMG_1234.jpg
IMG_1235.jpg

After:
kedar_photo_1.jpg
kedar_photo_2.jpg
